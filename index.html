Welcome to CS Zero!

Experience the full breadth Computer Science Program at the Friends School of Baltimore in just three weeks with CS Zero, our zero-experience, general-audience summer course. Available to everyone in person and online.

Enroll Today About the Program

Three Weeks

Week 1: Code and Computing

What is a computer? In this week, we'll explore the fundamentals of computing and coding including number systems, logical circuitry, and a basic theory of computation. We will also introduce the Python programming language and learn how to write your first of many programs.

Learn More
Week 2: Physical Computing

Break out of the laptop and explore how computers interact withthe physical world. Through hands-on lessons, we'll explore the basics of electronics using the Arduino microcontroller and a comprehensive variety of basic sensors, motors, and displays to create interactive products.

Learn More
Week 3: Creative Coding

Coding is a powerful tool for creative expression. Put your coding skills to the test by combining them with color, patterns, and interactivity. We'll explore ways to create both beautiful and engaging content using the Processing library.
